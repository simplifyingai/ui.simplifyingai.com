## About

The Lollipop Chart is a variation of a bar chart where bars are replaced with lines (stems) and circles (dots). It provides a cleaner look while maintaining the ability to compare values across categories.

## Usage

```tsx showLineNumbers
import { LollipopChart } from "@/components/ui/charts"

const data = [
  { category: "Jan", value: 45 },
  { category: "Feb", value: 52 },
  { category: "Mar", value: 78 },
]

export function Example() {
  return <LollipopChart data={data} orientation="horizontal" />
}
```

## API Reference

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| data | LollipopDataPoint[] | required | Array of data points |
| orientation | "horizontal" \| "vertical" | "horizontal" | Chart orientation |
| dotSize | number | 10 | Size of dots in pixels |
| showGrid | boolean | true | Show grid lines |
| showValues | boolean | false | Show value labels |
| color | string | "#2563eb" | Dot color |
| stemColor | string | "#93c5fd" | Stem color |

## Notes

- The lollipop chart is a cleaner alternative to traditional bar charts, reducing visual clutter
- It's particularly effective when displaying many categories or when comparing values
- The horizontal orientation is best for long category labels
- The vertical orientation works well for time-series data or short category names
- Dot size can be adjusted to emphasize or de-emphasize the data points
- Grid lines help with precise value reading and can be toggled on/off
- Value labels can be displayed directly on the chart for exact readings
- Custom colors for both dots and stems allow for brand consistency
- The minimal design makes it easier to focus on the actual data values
- Lollipop charts reduce ink-to-data ratio compared to bar charts while maintaining clarity
