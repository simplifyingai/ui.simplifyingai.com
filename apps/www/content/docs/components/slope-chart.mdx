## About

The Slope Chart connects two data points with a line to emphasize change over time or between two conditions. Lines are automatically colored green for increases and red for decreases, making it easy to spot trends at a glance.

## Usage

```tsx showLineNumbers
import { SlopeChart } from "@/components/ui/charts"

const data = [
  { category: "Marketing", start: 45, end: 72 },
  { category: "Sales", start: 68, end: 55 },
  { category: "Engineering", start: 32, end: 58 },
]

export function Example() {
  return (
    <SlopeChart
      data={data}
      startLabel="Q1 2024"
      endLabel="Q4 2024"
    />
  )
}
```

## API Reference

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| data | SlopeDataPoint[] | required | Array of data points |
| startLabel | string | "Before" | Label for start column |
| endLabel | string | "After" | Label for end column |
| showGrid | boolean | true | Show grid lines |
| showChange | boolean | true | Show percentage change |
| increaseColor | string | "#22c55e" | Color for increasing lines |
| decreaseColor | string | "#ef4444" | Color for decreasing lines |
| neutralColor | string | "#6b7280" | Color for unchanged lines |
| valueFormatter | function | toLocaleString | Format values |

## Data Format

```tsx
interface SlopeDataPoint {
  category: string  // Category or item name
  start: number     // Starting value
  end: number       // Ending value
  color?: string    // Optional: override auto color
}
```

## Notes

- Lines are automatically colored: green for increases, red for decreases
- Hover over any line to see detailed values and change amount
- Category labels appear on the left, end values with percentage change on the right
- Legend at top shows color meaning
- Best for comparing 3-10 categories between two time points
- Works well for before/after comparisons and period-over-period analysis
