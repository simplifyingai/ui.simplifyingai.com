## About

The Slope Chart connects two data points with a line to emphasize change over time or between two conditions. It's excellent for showing before/after comparisons, rankings changes, or performance differences.

## Usage

```tsx showLineNumbers
import { SlopeChart } from "@/components/ui/charts"

const data = [
  { category: "Product A", start: 45, end: 72 },
  { category: "Product B", start: 68, end: 55 },
  { category: "Product C", start: 32, end: 58 },
]

export function Example() {
  return (
    <SlopeChart
      data={data}
      startLabel="Q1 2024"
      endLabel="Q4 2024"
    />
  )
}
```

## API Reference

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| data | SlopeDataPoint[] | required | Array of data points |
| startLabel | string | "Before" | Label for start column |
| endLabel | string | "After" | Label for end column |
| showValues | boolean | true | Show value labels |
| showGrid | boolean | true | Show grid lines |

## Data Types

```tsx showLineNumbers
interface SlopeDataPoint {
  category: string  // Category or item name
  start: number     // Starting value
  end: number       // Ending value
  color?: string    // Optional: custom line color
}
```

## Notes

- Slope charts are most effective for comparing changes between exactly two time points or conditions.
- The steepness and direction of each line makes it easy to spot which categories improved, declined, or stayed stable.
- Color coding can be used to differentiate between positive and negative changes.
- For best readability, limit the number of categories to 10-15 items.
- This chart type works well for before/after comparisons, period-over-period analysis, and ranking changes.
- Unlike line charts, slope charts focus on the change itself rather than the path between points.
