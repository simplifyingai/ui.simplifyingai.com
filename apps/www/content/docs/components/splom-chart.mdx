## About

A SPLOM (Scatter Plot Matrix) displays all pairwise relationships between variables in a dataset. Each cell shows a scatter plot of two variables, with histograms on the diagonal showing each variable's distribution. Essential for exploratory data analysis and identifying correlations.

## Usage

```tsx showLineNumbers
import { SplomChart } from "@/components/ui/charts"

const data = [
  { id: "Item 1", values: { x: 10, y: 20, z: 15 }, group: "A" },
  { id: "Item 2", values: { x: 25, y: 15, z: 30 }, group: "B" },
]

const dimensions = ["x", "y", "z"]

export function Example() {
  return <SplomChart data={data} dimensions={dimensions} />
}
```

## API Reference

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| data | SplomDataPoint[] | required | Array of data points |
| dimensions | string[] | required | Variable names to include |
| cellSize | number | 100 | Size of each cell in pixels |
| pointRadius | number | 2.5 | Radius of scatter points |
| showLabels | boolean | true | Show dimension labels |
| showHistograms | boolean | true | Show histograms on diagonal |
| colorScheme | string[] | blue palette | Colors for groups |

## Data Types

```tsx showLineNumbers
interface SplomDataPoint {
  id: string                      // Unique identifier for the data point
  values: Record<string, number>  // Object with dimension names as keys
  group?: string                  // Optional: group for color coding
}
```

## Notes

- SPLOM charts are powerful for exploratory data analysis with multivariate datasets.
- The matrix shows each variable plotted against every other variable, revealing correlations and patterns.
- Diagonal cells typically show histograms or density plots of individual variable distributions.
- For best results, use 3-6 dimensions. More than 6 can make the chart difficult to read.
- The chart is symmetric around the diagonal - upper and lower triangles show the same relationships from different perspectives.
- Color-coded groups help identify clusters and patterns across multiple dimensions.
- Use the compact variant when you need to display more dimensions in limited space.
