## About

The Violin Chart combines box plots with kernel density estimation to show data distribution.

## Usage

```tsx showLineNumbers
import { ViolinChart } from "@/components/ui/charts"

const data = [
  { label: "Dataset A", values: [12, 15, 18, 22, 25, 28, 30, 32, 35] },
  { label: "Dataset B", values: [8, 12, 15, 20, 25, 30, 35, 40, 45] },
]

export function Example() {
  return <ViolinChart data={data} showBoxPlot showMedian />
}
```

## API Reference

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| data | ViolinChartData[] | required | Data with labels and values |
| showBoxPlot | boolean | true | Overlay box plot |
| showMedian | boolean | true | Show median point |
| bandwidth | number | auto | KDE bandwidth |

## Notes

- Violin charts combine box plots with kernel density estimation (KDE) to show the full distribution shape
- The width of the violin at each value represents the probability density at that point
- Ideal for comparing distributions across multiple categories or groups
- The `showBoxPlot` option overlays a traditional box plot showing quartiles and outliers
- The `showMedian` option highlights the median value with a distinct marker
- The `bandwidth` parameter controls the smoothness of the KDE curve - lower values show more detail but may be noisier
- Violin charts are particularly useful in scientific and statistical analysis where understanding the full distribution is important
