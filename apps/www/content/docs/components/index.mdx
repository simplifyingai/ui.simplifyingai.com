Simplifying AI Charts is a fully custom chart library built from scratch using D3.js for math/scales and pure SVG for rendering. No external charting library dependencies - just React, SVG, and Tailwind CSS.

## Chart Categories

### Basic Charts
- **Line Chart** - Multi-series line charts with curves
- **Bar Chart** - Vertical/horizontal, stacked/grouped
- **Area Chart** - Stacked area with gradients
- **Scatter Chart** - Bubble charts with multiple symbols
- **Pie Chart** - With labels and animations
- **Donut Chart** - With center content

### Statistical Charts
- **Histogram Chart** - Distribution histograms
- **Box Plot Chart** - Box and whisker with outliers
- **Violin Chart** - KDE density distributions

### Financial Charts
- **Candlestick Chart** - OHLC with volume
- **Waterfall Chart** - Cumulative changes
- **Funnel Chart** - Conversion funnels

### Scientific Charts
- **Heatmap Chart** - 2D heatmaps with color scales
- **Contour Chart** - Contour lines

### Specialized Charts
- **Radar Chart** - Spider/radar charts
- **Treemap Chart** - Hierarchical treemaps
- **Sunburst Chart** - Radial hierarchies
- **Sankey Chart** - Flow diagrams
- **Gauge Chart** - Gauges with thresholds

### Map Charts
- **Choropleth Chart** - Geographic maps with GeoJSON

## Installation

```bash
npx shadcn@latest add https://ui.simplifying.ai/r/chart.json
```

Then add specific chart types:

```bash
npx shadcn@latest add https://ui.simplifying.ai/r/line-chart.json
npx shadcn@latest add https://ui.simplifying.ai/r/bar-chart.json
```
