## About

Range Charts display data ranges using area bands or bar ranges. They're perfect for showing temperature ranges, confidence intervals, stock price ranges, or any data with low/high bounds.

## Usage

```tsx showLineNumbers
import { RangeChart } from "@/components/ui/charts"

const data = [
  { category: "Jan", low: -5, high: 8, mid: 2 },
  { category: "Feb", low: -3, high: 10, mid: 4 },
  { category: "Mar", low: 2, high: 15, mid: 8 },
]

export function Example() {
  return (
    <RangeChart
      data={data}
      lowLabel="Min"
      highLabel="Max"
      midLabel="Avg"
    />
  )
}
```

## API Reference

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| data | RangeDataPoint[] | required | Array of data points |
| variant | "area" \| "bars" | "area" | Visual style |
| showMidLine | boolean | true | Show middle value line/marker |
| showGrid | boolean | true | Show grid lines |
| fillColor | string | "#3b82f6" | Range fill color |
| strokeColor | string | "#1d4ed8" | Border stroke color |
| midLineColor | string | "#1e40af" | Mid-line color |
| lowLabel | string | "Low" | Label for low values |
| highLabel | string | "High" | Label for high values |
| midLabel | string | "Average" | Label for mid values |
| valueFormatter | function | toLocaleString | Format values |

## Data Format

```tsx
interface RangeDataPoint {
  category: string  // X-axis category label
  low: number       // Lower bound of range
  high: number      // Upper bound of range
  mid?: number      // Optional middle value (mean, median, etc.)
}
```

## Notes

- The `area` variant creates a smooth band between low and high values with a dashed mid-line
- The `bars` variant shows individual range bars with a solid mid-line marker
- Hover over any data point to see detailed values in the tooltip
- Legend automatically displays what the band and mid-line represent
- Use custom labels (lowLabel, highLabel, midLabel) to match your data context
