## About

Range Charts display data ranges using various visual styles including area bands, bar ranges, error bars, and floating bars. Perfect for showing temperature ranges, confidence intervals, stock price ranges, salary bands, or any data with low/high bounds.

## Usage

```tsx showLineNumbers
import { RangeChart } from "@/components/ui/charts"

const data = [
  { category: "Jan", low: -5, high: 8, mid: 2 },
  { category: "Feb", low: -3, high: 10, mid: 4 },
  { category: "Mar", low: 2, high: 15, mid: 8 },
]

export function Example() {
  return (
    <RangeChart
      data={data}
      lowLabel="Min"
      highLabel="Max"
      midLabel="Avg"
    />
  )
}
```

## API Reference

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| data | RangeDataPoint[] | required | Array of data points |
| variant | "area" \| "bars" \| "errorBars" \| "floating" | "area" | Visual style |
| orientation | "vertical" \| "horizontal" | "vertical" | Chart orientation |
| showMidLine | boolean | true | Show middle value line/marker |
| showGrid | boolean | true | Show grid lines |
| showValues | boolean | false | Show value labels |
| fillColor | string | "#3b82f6" | Range fill color |
| strokeColor | string | "#1d4ed8" | Border stroke color |
| midLineColor | string | "#1e40af" | Mid-line color |
| lowLabel | string | "Low" | Label for low values |
| highLabel | string | "High" | Label for high values |
| midLabel | string | "Average" | Label for mid values |
| valueFormatter | function | toLocaleString | Format values |

## Data Format

```tsx
interface RangeDataPoint {
  category: string  // X-axis category label
  low: number       // Lower bound of range
  high: number      // Upper bound of range
  mid?: number      // Optional middle value (mean, median, etc.)
  color?: string    // Optional custom color for this data point
}
```

## Variants

### Area
Smooth band connecting all data points. Best for time series data where you want to show continuous ranges like temperature bands or confidence intervals.

### Bars
Individual rectangular bars for each category. Ideal for comparing discrete ranges across categories like stock price ranges or salary bands.

### Error Bars
Statistical visualization with thin lines and caps. Perfect for scientific data, showing mean values with confidence intervals or standard deviations.

### Floating
Solid filled bars without borders. Great for cleaner visualizations of ranges like salary bands or price ranges where the focus is on the range itself.

## Notes

- All variants support both vertical and horizontal orientations
- The `area` variant only works with vertical orientation (horizontal falls back to bars)
- Hover over any data point to see detailed values in the tooltip
- Legend automatically displays what the band and mid-line represent
- Use custom labels (lowLabel, highLabel, midLabel) to match your data context
- Each data point can have a custom color via the `color` property
