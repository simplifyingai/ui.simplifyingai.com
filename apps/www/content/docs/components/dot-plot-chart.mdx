## About

The Dot Plot Chart displays individual data points as dots along a scale, making it easy to compare values across categories. It's a cleaner alternative to bar charts when you want to emphasize individual values.

## Usage

```tsx showLineNumbers
import { DotPlotChart } from "@/components/ui/charts"

const data = [
  { category: "Product A", value: 85 },
  { category: "Product B", value: 72 },
  { category: "Product C", value: 93 },
]

export function Example() {
  return <DotPlotChart data={data} orientation="horizontal" />
}
```

## API Reference

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| data | DotPlotDataPoint[] | required | Array of data points |
| orientation | "horizontal" \| "vertical" | "horizontal" | Chart orientation |
| dotSize | number | 12 | Size of dots in pixels |
| showGrid | boolean | true | Show grid lines |
| showValues | boolean | true | Show value labels |
| color | string | "#2563eb" | Default dot color |

## Notes

- Dot plots are ideal for comparing individual values across categories without the visual weight of bars
- They work best with 3-15 categories; too many categories make comparison difficult
- Horizontal orientation is recommended when category labels are long
- Grid lines help viewers accurately read values when `showGrid={true}`
- Unlike bar charts, dot plots emphasize precision over magnitude
- Each data point can optionally have its own color by adding a `color` property
