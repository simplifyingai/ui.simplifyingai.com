## About

The Dumbbell Chart (also known as a DNA chart or barbell chart) shows the change or difference between two data points. It's excellent for before/after comparisons, showing gaps between groups, or visualizing ranges.

## Usage

```tsx showLineNumbers
import { DumbbellChart } from "@/components/ui/charts"

const data = [
  { category: "Marketing", start: 45, end: 72, startLabel: "2023", endLabel: "2024" },
  { category: "Sales", start: 68, end: 85, startLabel: "2023", endLabel: "2024" },
]

export function Example() {
  return <DumbbellChart data={data} orientation="horizontal" />
}
```

## API Reference

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| data | DumbbellDataPoint[] | required | Array of data points |
| orientation | "horizontal" \| "vertical" | "horizontal" | Chart orientation |
| dotSize | number | 10 | Size of dots in pixels |
| showGrid | boolean | true | Show grid lines |
| showValues | boolean | true | Show value labels |
| startColor | string | "#93c5fd" | Color for start dots |
| endColor | string | "#1e40af" | Color for end dots |
| connectorColor | string | "#60a5fa" | Color for connecting line |

## Notes

- Dumbbell charts excel at showing before/after comparisons, growth/decline, or gaps between groups
- The connecting line makes it easy to see the magnitude of change at a glance
- Use contrasting colors for start and end points to clearly distinguish them
- Horizontal orientation is recommended when category labels are long
- Works best with 3-12 categories for optimal readability
- Consider using different color schemes to indicate positive/negative changes
- The `startLabel` and `endLabel` properties provide context for what each endpoint represents
