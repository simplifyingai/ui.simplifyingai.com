title: Simplifying AI UI
description: A comprehensive chart library built on shadcn/ui
---

Simplifying AI UI is a component library and custom registry built on top of [shadcn/ui](https://ui.shadcn.com/) to help you build beautiful, interactive data visualizations. It provides 40+ chart types built with React, SVG, and D3.js for math calculations.

Components are available via the `shadcn` CLI:

```bash
npx shadcn@latest add https://ui.simplifying.ai/r/<component>.json
```

For example, to install the [Funnel Chart](/docs/components/funnel-chart), you can run:

```bash
npx shadcn@latest add https://ui.simplifying.ai/r/funnel-chart.json
```

## Chart Categories

### Basic Charts
- [Line Chart](/docs/components/line-chart) - Time series and trend visualization
- [Bar Chart](/docs/components/bar-chart) - Categorical comparisons
- [Area Chart](/docs/components/area-chart) - Stacked area and trends
- [Pie Chart](/docs/components/pie-chart) - Part-to-whole relationships
- [Donut Chart](/docs/components/donut-chart) - Ring charts with center content
- [Scatter Chart](/docs/components/scatter-chart) - Point distributions

### Statistical Charts
- [Histogram Chart](/docs/components/histogram-chart) - Distribution analysis
- [Box Plot Chart](/docs/components/box-plot-chart) - Statistical summaries
- [Violin Chart](/docs/components/violin-chart) - Density distributions

### Financial Charts
- [Candlestick Chart](/docs/components/candlestick-chart) - OHLC data
- [Waterfall Chart](/docs/components/waterfall-chart) - Cumulative changes
- [Funnel Chart](/docs/components/funnel-chart) - Conversion funnels

### Scientific Charts
- [Heatmap Chart](/docs/components/heatmap-chart) - 2D intensity maps
- [Contour Chart](/docs/components/contour-chart) - Density contours

### Specialized Charts
- [Radar Chart](/docs/components/radar-chart) - Multi-variable comparison
- [Treemap Chart](/docs/components/treemap-chart) - Hierarchical data
- [Sunburst Chart](/docs/components/sunburst-chart) - Nested hierarchies
- [Gauge Chart](/docs/components/gauge-chart) - KPI indicators

### Map Charts
- [Choropleth Chart](/docs/components/choropleth-chart) - Geographic data
